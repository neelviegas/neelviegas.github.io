Local AI Model

Personal Project
Role: Personal Project Tools: Ollama 3.2 · RAG · Python Timeline: Jun 2026 – present

Goal: Learn about AI and current use cases, and build a tool to help with thermodynamic questions (Thermodynamics 2 & Rocket Team).

After working on a local AI at Pratt and Whitney, I was curious and decided to try building one myself. So far, I have built a retrieval-augmented generation pipeline running fully offline via Ollama, using a Qwen 32B model to answer thermodynamics questions grounded in "Thermodynamics for Engineers version 7" (1000 pages). I wrote three Python components: an ingestion script (chunking/embedding the source text), a main orchestration script, and a query script triggered by a custom solve command syntax that retrieves relevant passages and generates cited answers.

Currently I am trying to improve the solving by optimizing the CPU/GPU workload for my setup (GPU: RX 5700 XT, CPU: Ryzen 7 5800X). The parameters also need to be refined further for better performance.

Thrust Analysis Tool

McGill Rocket Team
Role: Injector team lead Tools: Python · Matplotlib · NumPy Timeline: May 2026 – present

Goal: A python pipeline that takes in hotfire test data and outputs the coefficient of combustion efficiency.

It takes in raw Hotfire data from our hybrid rocket engine and spits out the combustion efficiency (and some pretty graphs). The main difficulties revolved around making assumptions to deal with faulty sensors, and deciding if I could trust the data.

  • Results: c* efficiency avg = 85.3%
  • Used Python Matplot and NumPy to get to the final product

Automatic Curtains

Personal Project
Role: Personal Project Tools: SolidWorks · Arduino UNO · 3D print Timeline: Jun 2026 – present

Goal: I live next to a building whose lights, unfortunately for me, never turn off. That said, I like waking up to the sun without compromising my sleep. The goal was therefore to build curtains that open on a timer.

Custom gear train and servo mount designed in SolidWorks. Powered and controlled by an Arduino UNO that opens the curtains in the morning and closes them at night.

  • Result: Fully assembled product.
  • Designed the drive gears and servo/electronics mounts in SolidWorks, then 3D printed them.
  • Built the circuit around an Arduino UNO with a timed open/close routine.
  • Iterated on the gear set until the torque and fit were right.
